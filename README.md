## Description

[Hapi](https://github.com/hapijs/hapi) project to vote for your favorite Pokemon.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development in watch mode
$ npm run start
```

## Test

There are no tests written for this project.

## API Documentation
### Swagger

```bash
# development
$ npm run start
```
